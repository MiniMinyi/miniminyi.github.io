用github pages和hexo来建个人网站

2月 06, 2016

列一下参考过的网站。

  1. Zippera’s blog
    其实只看这个blog的这个list就能顺利完成买域名，用github pages写blog，学习markdown简单用法，在网页中添加音乐、图片等工作。
  2. github pages
  3. hexo官方文档
  4. hexo各种主题
  5. hexo更多的主题
  6. 七牛云

然后在这里简单地说一下我的建站步骤。

  1. 在github上新建一个miniminyi.github.io的repository
  2. 在godaddy上租了两年miniminyi.com的域名
  3. 绑定域名和github pages
  4. 挑了一个喜欢的主题Tranquilpeak,然后修改了一下个人简介，enable了tag和archive。
  5. 在七牛注册了账号，把图片上传
  6. 写了第一篇blog来熟悉markdown的语法和尝试七牛图库的速度

真的是很简单呢~
